Sigma YAML → SigmaJSON Converter (Offline)

Converts Sigma YAML into SigmaJSON compatible with your browser triage engine. Supports --- separated rule packs, multi-selection, ATT&CK tags, and validation warnings.

1) Input Sigma YAML (one rule or multiple separated by ---)

Supported: title, id, level, tags, logsource.service, detection.selection*, and detection.condition using basic and/or. Unsupported Sigma features will generate warnings.

2) Validation Warnings

None

3) Output SigmaJSON

{}

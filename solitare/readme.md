# Patience Deck - A Solitaire Game Collection

## Description

Patience Deck is a simple web-based solitaire game that currently supports two classic variations: Klondike (Classic) and Golf Solitaire.  It allows users to select a game from a dropdown menu and play using drag-and-drop functionality.  The game tracks score and moves, displaying them on the screen.

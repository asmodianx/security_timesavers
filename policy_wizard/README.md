# NIST Documentation Wizard v7

This version removes the editable **Example** field from the section editor. Examples remain as static guidance from the selected template, but they are not user data-entry fields.

## Data Entry

Open `nist_documentation_wizard_v7.html` and go to **Step 5: Sections**.

Each section has these editable fields:

- Section Title
- Instruction
- Content
- Section References
- Section Notes

The **Content** field is the section body used by the live preview, Markdown export, JSON export/import, and browser Print / Save as PDF.

## Features

- Standalone HTML/JavaScript
- No CDN
- No external libraries
- No network calls
- Policy, Procedure, Standard, and Best Practice templates
- Static examples from templates, not editable example fields
- Add, duplicate, delete, rename, and reorder sections
- Revision tracking and snapshots
- NIST CSF / SP 800-53 / RMF alignment fields
- Markdown export
- JSON export/import
- Browser Print / Save as PDF

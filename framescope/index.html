FrameScope - Standalone Iframe Sandbox Viewer

OWASP-hardened local iframe wrapper focused on browser-observable behavior and iframe boundary controls. Credentialless is incognito-like isolation, not true Incognito mode.

Navigation

Scope: FrameScope can configure iframe boundary restrictions and log frame load behavior. It cannot put an iframe into true browser Incognito mode or observe a remote cross-origin page's internal API attempts.

Sandbox controls

Feature permissions / iframe allow policy

Checked features are allowed only for the iframe target source using feature 'src'. Unchecked features are denied using feature 'none'. Target-site headers, browser prompts, and user choice may still block allowed features.

Other iframe boundary controls

Credentialless note: This is incognito-like cookie/storage isolation where supported. It is not true browser Incognito mode. Browser support varies.
Strict iframe CSP note: May break modern websites. Useful for maximum-isolation rendering tests.

Maximum isolation: enables credentialless, strict iframe CSP/no-script mode, no referrer, deny-all features, and an empty sandbox attribute.
